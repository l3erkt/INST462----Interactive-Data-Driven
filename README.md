# 🎧 What Makes a Hit Song?

An interactive, data-driven web experience exploring the **Spotify All-Time Top 100 Songs**. This project investigates whether musical features like BPM, danceability, energy, and emotional tone actually influence a song’s streaming success.

---

## 📊 Project Overview

This project is a **scrollytelling data visualization** that allows users to explore patterns behind some of the most streamed songs ever.

Users can interact with multiple visual sections to answer key questions:

- Does tempo (BPM) impact popularity?
- Are more danceable songs more successful?
- Do happier songs get more streams?
- Which genres dominate streaming charts?

---

## ✨ Features

### 1. 🎤 The Stage (Genre Exploration)
- Filter songs by genre  
- Visual “crowd” representation of songs  
- Larger dots = more streams  

Displays:
- Total streams  
- Song count per genre  

---

### 2. 🎚️ BPM Dial (Tempo Analysis)
- Interactive slider to adjust BPM (0–180)  
- Shows which genres cluster around selected tempo  

Highlights:
- Average BPM per genre  
- Total streams  
- Number of songs  

---

### 3. 📈 Scatter Plot (Feature vs Streams)
- Compare audio features against total streams:
  - Danceability  
  - Energy  
  - Valence (mood)  
  - Acousticness  
  - BPM  

Includes:
- Genre filtering  
- Trend line (linear regression)  
- Interactive tooltips  

---

### 4. 😊 Emotion Factor (Valence)
- Explore emotional tone of songs  
- Visual face reacts to mood (valence)  

Displays:
- Average valence per genre  
- Emotional interpretation  
- Stream impact
